# Dossier images

Placez ici vos images pour faciliter les modifications avant hébergement.

Exemples de noms recommandés:

- `welcome.jpg` (fond d'accueil)
- `hero.jpg` (fond principal du titre)
- `photo-1.jpg` a `photo-12.jpg` (galerie)

Ensuite, utilisez la page `pages/personnalisation.html` pour mettre à jour les chemins ou importer automatiquement.
