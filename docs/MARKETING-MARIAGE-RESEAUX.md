## Parcours démo à filmer (enregistrement écran)

1. **Porte d'entrée** — saisir un prénom invité → « Bienvenue [Prénom] »
2. **Enveloppe** — ouverture animée · message *« Avec amour, nous vous ouvrons cette enveloppe de bonheur »*
3. **Invitation** — noms des mariés, date, lieu
4. **Programme** — horaires (arrivée, entrée mariés, danses…)
5. **GPS / lieu** — carte et adresse de la salle
6. **RSVP** — « Confirmer ma présence » → formulaire → confirmation
7. **QR code** — badge invité après confirmation
8. **Bonus** (optionnel) — galerie, livre d'or, menu boissons, dress code
